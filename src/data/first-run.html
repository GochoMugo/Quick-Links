Quick Links

Addon for Firefox : Lets you add links to your favorite sites to your add-on bar from where you can quickly access the sites

Specifications

Compatability Firefox 21.0a - 32.0
Languages English (United States), Swahili

Icons

Lets you add icons to your addon bar. The icons serve as links on which you can link to open a new tab with the URL address loading.

Preview

Presets

The addon offers some presets of links that can be added to the addon bar. Some presets are:

  • Facebook
  • LinkedIn
  • Twitter

Preview

Custom links

With this addon, you can add a link of your choice to the addonbar.

Preview

Hot Keys

Ctrl / Cmd + Alt + Q lets you add links quickly. The panel will be shown will the input boxes for title and url already filled with your current tab's title and url respectively.
